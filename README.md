# react-cli

#### 项目介绍
react-cli

#### 软件架构
一个基于webpack、webpack-server-dev搭建的react项目;
使用react-router-dom、redux、redux-react技术，并对源码分析，手动实现;
涉及登录拦截，路由传参，http服务搭建

#### 安装教程

1. 打开git
2. 运行 git clone https://github.com/lcp-xj/react-redux-explain.git
3. 或点击下载ZIP后解压

#### 使用说明

1. cd 项目目录
2. npm install
3. npm run dev
